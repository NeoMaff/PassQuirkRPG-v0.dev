# 📝 Registro de Cambios - PassQuirk RPG

## [v0.4.0] - 2025-03-11
### ✨ Añadido
- Nuevo comando `/admin` para alternar entre modo administrador y usuario normal

- Tutorial interactivo con combate contra Slime
- Sistema de botones y menús en todos los comandos principales

- Actualizada la estructura del bot según la documentación

- El comando `/start` ahora es solo para pruebas en modo admin

### 🐛 Corregido

- Actualizado el sistema de niveles y experiencia
